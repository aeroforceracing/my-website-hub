# AeroForce Racing

Official website of **AeroForce Racing** — a young engineering team competing in F1 in Schools (STEM Racing).

Engineering the future, one race at a time. Precision. Innovation. Speed.

## Tech Stack

- Vite + React 18 + TypeScript
- Tailwind CSS + shadcn/ui
- Framer Motion
- React Router

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

Deployed on Vercel.
